<h3>Rules Interpretations</h3>
Some bastion rules required interpretation on my part, so here are some decisions I made that are not explicitly spelled-out in the rules, organized by facility.

Armory
- There are no rules for a partially-stocked armory, in the case of a recruit order after a trade order to restock. For such instances I have implemented partial defense improvement. A number of dice will be d8s proportional to the amount of stock in the armory to the total number of defenders.
